NEW Fastbridger NPC UI + Incline/Short Bridger Added

  • Thread starter Zackaria
  • Start date Jun 22, 2021
  • Jun 22, 2021
  • Replies: 12
Good morning or evening!

For a few days now, we have been working on adding new things for you guys! One of the things the higher ups spoke about was adding more to our Fastbridger modes, and today we've decided that we have done enough testing and are deciding to release it today!

What we've added:
Diagonal (incline) bridge mode: This is basically for players who want to practice diagonal god bridging and or diagonal speed bridging
Short bridge mode: A shorter version of our original Fastbridger for practicing short distance bridging rather than long bridging. Mainly for beginners, but anyone can use it

Another thing we've re-worked is the ingame GUI while selecting the NPC's which are only available in the Short & Regular mode as of now


We hope to have made you happy with these updates
Join our discord: https://discord.gg/bridge for more updates!

Zackaria
